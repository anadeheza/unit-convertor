# Unit Converter

Just a simple unit converter made with Python in a simple website made with HTML, CSS and a bit od js

## Units
You can convert:

Length units, including milimeters, centimeters, meters and kilometers, 

Weight units, like miligrams, centigrams, grams and kilograms

Data size units, such as bits, bytes, kilobytes, megabytes, gigabytes and terabytes

## How to use
The site is pretty straightforward, just choose the category (length , weight ot data size) and select the unit you want to convert from and the one you wanna convert to, finally, add the value you wanna use and press "convert"

The result will appear right at the bottom of the card ;)
